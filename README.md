# Premium Web Developer Portfolio

A stunning, responsive, and highly interactive portfolio website built with modern web technologies. This project showcases a premium design aesthetic with smooth animations, glassmorphism effects, and a custom dark theme.

## âœ¨ Features

- **Premium UI/UX**: Modern dark theme with vibrant gradients and glassmorphism.
- **Interactive Hero Section**: Dynamic background with animated gradient orbs and staggered text reveals.
- **Smooth Navigation**: Glassmorphism navbar with scroll detection and buttery smooth anchor links.
- **Animated Sections**: 
  - **About Me**: Professional bio and animated stats cards.
  - **Skills**: Technology grid with animated proficiency bars.
  - **Projects**: Featured project cards with gradient headers and elevation effects.
  - **Contact**: Functional lead generation form and social connectivity.
- **Responsive Design**: Optimized for everything from mobile phones to large desktop monitors.
- **Performance**: Built with Vite and TypeScript for lightning-fast speeds and reliable development.

## ğŸš€ Getting Started

### Prerequisites

- Node.js (Version 18+ or 20+ recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Saurabhpaliwal2/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the local development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Production

Build the project for production:
```bash
npm run build
```
The output will be in the `dist/` folder.
